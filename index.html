TikTok KPI Dashboard DZD

Internal analytics dashboard for tracking TikTok video performance metrics.

What this app does

This web application connects to a TikTok account (with user authorization) to retrieve video performance metrics such as views, likes, shares, and comments. The data is used for internal analytics and reporting.

How data is processed

The application uses TikTok APIs to retrieve authorized data, automation workflows (n8n) to schedule collection, and Microsoft 365 (Excel Online / OneDrive) for internal storage and reporting.
